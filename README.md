# Sales_Data_Dashboard
This Sales Performance Dashboard highlights key metrics like revenue, profit, units sold, and orders. It visualizes item performance, profit vs. revenue, and sales channels, with interactive filters for region, country, order priority, and item type for in-depth analysis.

<h2>Project Overview</h2>
<p>This project aims to analyze a diverse set of global sales records to identify key revenue drivers, assess profitability across different regions and item types, and evaluate the performance of various sales channels. The goal is to provide a holistic view of the company's financial health and sales dynamics for strategic planning.</p>
